什么样的源代码适合阅读

一般是“麻雀虽小五脏俱全”的项目适合阅读，在这里 c/c++ 方面的可以推荐几个供读者参考：

  • tinyxml
  • cJSON
  • libevent
  • memcached
  • redis
  • leveldb
  • nginx
  • lua

python web 框架方面：

  • flask
  • django

javascript 方面：

  • jquery
  • vue

这些都能在网络上找到一手的源码，有兴趣的可以试试。
